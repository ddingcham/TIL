Out - In 방식으로 접근하려 했더니 생각보다 전체 OutLine 잡는 게 어려워서  
꼭 필요하지만 작은 몇 개의 요구사항 구현을 먼저 진행하기로 함  
구현한 작은 요구사항들(In)을 토대로 Out을 수정 하고, 
전체 OutLine이 수정될 때마다 지속적으로 리팩토링 작업을 하기로 함
